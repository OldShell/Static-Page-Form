The Advanced Encryption Standard (AES) is a U.S. Federal Information Processing Standard (FIPS). It was selected after a 5-year process where 15 competing designs were evaluated. MS Word uses AES-128 up to 2013 and AES-256 thereafter, this system (CryptoJS) uses AES-256. However this implementation is in JavaScript and there could be breaches there. Malware could compromise the system at the encryption / decryption stage. In my opinion the level of security is still way in excess of what we require, but not quiet the no-brainer I was hoping for. Perhaps the question "What level of security do we REALLY need?" still has to be answered.
